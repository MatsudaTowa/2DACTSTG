◆このゲームについて
刃がない剣士が柄のみで攻撃したり、特殊な斬撃を使い戦う横スクロールアクションゲーム

◆起動方法
フォルダ内のexeファイルを起動することでゲームスタート

◆基本操作
7/4時点:キーボードとマウスのみ

・キーボード
Aキー:左移動
Dキー:右移動
SPACEキー:ジャンプ(二段ジャンプ可能)
ENTERキー:画面遷移

・マウス
左クリック:斬撃発射(ゲージある時のみ)
右クリック:近接攻撃


◆実装内容
・プロトタイプ版時点(6/28)
プレイヤーとオブジェクトの当たり判定
プレイヤーの必須の動き(移動、ジャンプ、斬撃発射)
画面遷移
動くだけの敵

・Reプロトタイプ版時点(7/4)
プレイヤーの近接攻撃追加
敵とオブジェクト、プレイヤー、プレイヤーの攻撃の当たり判定
ゲージを軽く実装
